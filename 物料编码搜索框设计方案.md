**版本**：V1.0

## 一、功能目标

做一个单框智能搜索框，用户输入物料编码或名称，实时显示匹配结果，支持任意层级搜索，选中后显示完整路径和子级编码。

---

## 二、数据准备

### 2.1 数据源

Excel 文件 `物料编码分级表.xlsx`，包含 4 个子表：

| 子表名称 | 行数 | 编码格式 |
|---------|------|---------|
| 一级编码 | 47 条 | 2 位数字 |
| 二级编码 | 616 条 | 4 位数字 |
| 三级编码 | 3388 条 | 6 位数字 |
| 四级编码 | 279 条 | 8 位数字 |

**总计**：约 4330 条物料数据

### 2.2 数据结构

每条物料包含以下字段：

```javascript
{
  code: "01010201",        // 物料编码
  name: "菱形花纹钢板",     // 物料名称
  level: 4,                // 层级 (1-4)
  parentCode: "010102",    // 父级编码 (一级编码为 null)
  children: []             // 子级编码列表 (四级编码为空数组)
}
```

### 2.3 数据处理步骤

1. **读取 Excel**：读取 4 个子表，合并成一个扁平的物料列表
2. **构建树形结构**：每个节点包含 `children` 数组，用于快速查找子级
3. **建立索引**：
   - **编码索引**：编码 → 物料对象（用于精确查找）
   - **名称索引**：名称关键词 → 物料列表（用于模糊搜索）

## 三、搜索框核心逻辑

### 3.1 输入识别规则

| 输入类型 | 匹配规则 | 示例 |
|---------|---------|------|
| 2 位数字 | 一级编码前缀匹配 | 输入"01"显示所有一级编码 |
| 4 位数字 | 二级编码前缀匹配 | 输入"0101"显示 0101 及其所有子级 |
| 6 位数字 | 三级编码前缀匹配 | 输入"010102"显示 010102 及其所有子级 |
| 8 位数字 | 精确匹配四级编码 | 输入"01010201"精确返回菱形花纹钢板 |
| 中文/字母 | 名称模糊匹配 | 输入"花纹"匹配所有名称含"花纹"的物料 |
| 拼音首字母 | 拼音匹配 | 输入"hgb"匹配"花纹板" |

### 3.2 搜索算法

```javascript
function search(query) {
  if (isNumeric(query)) {
    // 数字输入：按编码前缀匹配
    return materials.filter(m => m.code.startsWith(query))
  } else {
    // 中文输入：按名称模糊匹配
    return materials.filter(m => 
      m.name.includes(query) || 
      pinyinMatch(m.name, query)
    )
  }
}
```

### 3.3 搜索结果处理

1. 搜索结果按层级排序（一级优先，四级在后）
2. 最多显示 50 条结果，避免下拉框过长
3. 用户选中某条物料后，显示该物料的完整路径和所有子级编码

## 五、关键交互细节

| 功能 | 说明 |
|------|------|
| **实时搜索** | 用户输入时即时显示结果，无需点击搜索按钮 |
| **键盘导航** | 支持上下箭头选择结果，回车确认选中，ESC 关闭下拉框 |
| **点击外部关闭** | 点击搜索框外部区域时关闭下拉框 |
| **空状态处理** | 无匹配结果时显示"未找到相关物料，请尝试其他关键词" |
| **加载状态** | 数据加载时显示 Loading 动画 |
| **路径面包屑** | 选中物料后，完整路径支持点击跳转（如点击"0101 板材"可切换到该层级） |
| **一键复制** | 详情区域提供"复制编码"和"复制名称"按钮 |

### 6.3 核心函数伪代码

```javascript
// 获取完整路径
function getFullPath(material) {
  const path = []
  let current = material
  while (current) {
    path.unshift(current)
    current = findMaterialByCode(current.parentCode)
  }
  return path
}

// 获取子级编码
function getChildren(material) {
  return materials.filter(m => m.parentCode === material.code)
}

// 拼音匹配
function pinyinMatch(name, query) {
  const pinyin = getPinyinFirstLetters(name)
  return pinyin.toLowerCase().includes(query.toLowerCase())
}
```

## 七、性能优化建议

| 优化项 | 说明 |
|--------|------|
| **数据预加载** | 页面加载时一次性读取所有物料数据到内存 |
| **防抖处理** | 输入框添加 300ms 防抖，避免频繁搜索 |
| **结果限制** | 搜索结果最多显示 50 条，避免下拉框过长 |
| **虚拟滚动** | 如结果过多，使用虚拟滚动优化性能 |
| **缓存机制** | 缓存用户搜索过的关键词和结果 |
